How to compile file into executable:
1. Type make into the command line to compile coin_change.c as an executable called project2.
2. The executable requires a 2nd argument, CoinTest.txt file. 
3. Type project2 CoinTest.txt to start program.
4. Follow prompt on screen to continue with program output.
